= riff CLI tool

* link:riff/docs/riff.md[Documentation]

built with https://github.com/spf13/cobra[cobra]

== Building
```
go get -d github.com/projectriff/riff-cli/...
cd $(go env GOPATH)/src/github.com/projectriff/riff-cli/riff
make
```

== Running tests
```
cd $(go env GOPATH)/src/github.com/projectriff/riff-cli/riff
make test
```
